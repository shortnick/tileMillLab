      Process Step:
        Process Description:
          The Microstation-format file "railall.dgn" was converted to measuring units of feet using Intergraph's "wu_setup" command, and the resulting file was converted to Arc/Info coverage format with the Arc command "IGDSARC." The resulting coverage was transformed to geographic coordinates with Arc commands "PROJECTDEFINE" and "PROJECT" and ownership attributes were assigned based on the Intergraph color code found in INFO's "ACODE" table. Uncoded lines were assigned owner values in ArcEdit.
